# gitscratch-backend-flask

GitScratch 的社区后端。

## 安装依赖 / Installing Requirements
```shell
pip install -r requirements.txt
```

## 启动 / Start

```shell
# 如果你是Linux：
sudo sh ./start_prod.sh
#对于Windows:
sudo sh ./start_prod.cmd
```

## 调试 / Debugging
特别的，如果你想以开发环境运行：
```shell
# 如果你是Linux：
sudo sh ./start_dev.sh
#对于Windows:
sudo sh ./start_dev.cmd
```
